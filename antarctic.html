aligned the reads in all Antarctic runs against the RSRS version of the human mtDNA reference genome, I did variant calling for all runs except SRR13441706 pooled together, I ignored mutations with MAPQ below 100, and I selected the haplogroups in the PhyloTree mtDNA tree which had the highest number of shared mutations. There were 4 different mtDNA haplogroups which were tied on first place with 20 shared mutations: M7b1a1b (which is found at Yfull in China, Vietnam, Thailand, Laos, Kyrgysztan, and Philippines), M1a5 (which is found in Yemen, Kenya, Ethiopia, Somalia, and Tanzania), M1a3b1 (which is found in Spain, Italy, Morocco, Algeria, and Russia), and M10a1a1b1 (which is found in China, Japan, and Russia). So the reads might contain a mixture of different human mtDNA haplogroups, and you can't probably tell that much based on the analysis that Csabai et al. did using mixemt and 1000 Genomes. - mongol-fi
